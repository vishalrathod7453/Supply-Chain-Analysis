# Supply Chain Data Analysis and Prediction

## Project Overview
This project analyzes supply chain operations and predicts future demand using data analytics and machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
Supply Chain Data Analysis and Prediction.csv

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building
5. Prediction and Evaluation
6. Visualization

## Results
- Improved demand forecasting
- Identified supply chain bottlenecks
- Generated business insights through dashboards
